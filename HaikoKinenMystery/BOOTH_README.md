# 南谷小学校 閉校記念アーカイブ Booth版

これは架空のWeb横断型モキュメンタリー謎解きです。
実在の学校、人物、事件、団体とは関係ありません。

## はじめかた

フォルダ内の `web/index.html` をブラウザで開いてください。

推奨:

- PCブラウザ
- メモを取れる環境
- プレイ時間 2から3時間
- 1から3人プレイ

## 遊び方

南谷小学校の閉校記念アーカイブ、ニュース、卒業生ブログ、ガラケー風ページ、留言板などを巡回し、資料の矛盾を探してください。

答えが分かったら `web/investigate/index.html` の調査ノートに入力します。
調査ノートは15段階です。前の段階を解くと次の段階が開きます。

総合入口と調査ノートは開いたままにし、学校サイトやニュースなどの作中サイトは別タブまたは別ウィンドウで確認する想定です。

## 内容物

- `web/index.html`: ゲーム入口
- `web/school/`: 南谷小学校の閉校記念サイト
- `web/news/`: 外部ニュース風サイト
- `web/blog/`: 卒業生ブログ風サイト
- `web/mobile/`: 2006年頃の児童作成ガラケー風ページ
- `web/board/`: 卒業生留言板
- `web/investigate/`: 資料照合ノート
- `web/assets/`: ChatGPT生成画像
- `server.mjs`: ローカルサーバー起動用ファイル

現在の収録規模:

- HTML 43ページ
- PNG画像 8点
- 夜間巡回控え、閲覧制限資料、携帯メール控え、写メ日記、留言板の夜カテゴリを追加
- 校内図、台紙裏、展示ケース、補修写真控えなどの画像資料あり

## 注意

- 画像素材はChatGPT生成画像です。
- 今回追加した証拠風画像も、加工素材ではなくChatGPTで新規生成した画像です。
- スプライトシートは使用していません。
- `web/mobile/index.html` は当時風の再現のため、CSSを使わずHTMLのみで表現しています。
- ブラウザの安全設定によってローカルファイルの挙動が異なる場合は、同梱の `server.mjs` を使ってローカルサーバーで開いてください。

## ローカルサーバーで開く場合

Node.jsが使える環境では、フォルダ内で以下を実行します。

```powershell
node server.mjs
```

その後、ブラウザで以下を開きます。

```text
http://127.0.0.1:4381/web/
```
