# 南谷小学校 H18年度記録

販売想定のモキュメンタリー型Web謎解きです。プレイヤーは、匿名で届いた未整理アーカイブを検証する協力者として調査を開始します。

## 導入
閉校記念アーカイブの公開直前、卒業生を名乗る人物から未整理の学校資料が送られてきた。

依頼内容: 記念サイトに残らない児童の名前が、なぜ複数の資料でだけ現れるのかを確認してください。

## プレイ情報
- 想定時間: 60〜120分
- 推奨人数: 1〜3人
- 入口: `web/index.html`
- 形式: 架空Webサイト横断、資料照合、段階式の調査ノート

## この作品の特徴
事件名を前面に出さず、閉校記念資料の違和感から始まる学校アーカイブ型。懐かしさの中に不在の名前が浮く。

## 添付資料
- 学校資料室
- 携帯連絡網
- 留言板
- 調査ノート
- PTA記録
- 閉校記念ニュース

## 最終的に確定すること
記録の空白が相沢澪を6年1組周辺へ誘導し、連絡網メモを使って校舎裏の導線から退路を塞いだ。
