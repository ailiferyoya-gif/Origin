INVESTIGATION NOTE

資料照合ノート

このページは作中サイトではなく、プレイヤーが資料を照合するための調査UIです。学校サイト、新聞、ブログ、携帯向け資料は別ウィンドウで開き、このノートに戻って回答を入力してください。

未解放のカードは詳細が表示されません。前の段階を解くと、次の照合式と入力欄が開きます。ひとつのページだけで答えを決めず、同じ表記が別資料でどう扱われているかをメモしてください。

第1問: 写真にいない児童

同じ学級を扱う資料で、人数の扱いが揃っていない箇所を探します。

照合式学級資料の名簿欄 + 同窓生の記憶
入力児童名

第2問: 名簿登録数

台紙の説明と名簿欄を見比べ、登録された側の数を拾います。

照合式学級台紙 + 名簿登録欄
入力人数
第1問を解くと有効になります。

第3問: 写真人数

保存名に残された数は、登録数とは別のものを示しています。

照合式写真の保存名 + 登録数との差
入力写真側の人数またはファイル名
第2問を解くと有効になります。

第4問: 閉じられた日

安全連絡と貸出記録が同じ日に重なる箇所を確認します。

照合式校内の閉鎖記録 + 図書室の未完了記録
入力日付
第3問を解くと有効になります。

第5問: 保管箱番号

複数の展示箱のうち、封の状態と未完了資料が結びつくものを選びます。

照合式未完了の貸出 + 展示箱の状態
入力保管箱番号
第4問を解くと有効になります。

第6問: 未返却資料1

当日の天候と結びつく未完了資料を、貸出欄から拾います。

照合式貸出カード + 当日の天候記録
入力資料名
第5問を解くと有効になります。

第7問: 未返却資料2

もうひとつは、修繕記録と対になる資料です。

照合式貸出カード + 工事台帳
入力資料名
第6問を解くと有効になります。

第8問: 場所

安全連絡、工事記録、学校だよりに同じ場所の言い換えが出ます。

照合式危険箇所の連絡 + 工事記録 + 配布文面
入力場所
第7問を解くと有効になります。

第9問: 現地確認者

連絡した人と、現場へ行った人が同じ欄に残っています。

照合式安全委員会メモ + 当日の対応欄
入力人物名
第8問を解くと有効になります。

第10問: 工事業者

工事台帳と翌年度の御礼文で同じ名前を探します。

照合式補修記録 + 御礼文
入力業者名
第9問を解くと有効になります。

第11問: 留言板の持ち物

目撃談のなかで、持ち主と持っていた人がずれているものを探します。

照合式目撃投稿 + 児童用品の記述
入力持ち物
第10問を解くと有効になります。

第12問: 空白の確認時刻

帰宅確認の表で、ほかと同じ処理になっていない欄の時刻を拾います。

照合式確認表 + 空白欄
入力時刻
第11問を解くと有効になります。

第13問: 翌日の修正語

空白だった欄は、翌日に別の扱いへ置き換えられています。

照合式確認表の追記 + 名簿メモ
入力修正された言葉
第12問を解くと有効になります。

第14問: 不自然な配布日

本文の季節と台帳の日付が合わない号を探します。

照合式本文時期 + 配布台帳
入力台帳上の配布日
第13問を解くと有効になります。

第15問: 真相

これまでの人物、空白、危険箇所、修正記録をひとつの出来事として説明します。

照合式現地確認者 + 消えた児童 + 危険箇所 + 記録改変
入力誰が、誰を、どこで、どうしたのか
第14問を解くと有効になります。

南谷小学校アーカイブの真相

平成18年度6年1組の相沢澪は転校していません。大雨の日、旧体育館東階段で工事の不備を記録しようとしていました。

PTA会長の橘修司は、鳴瀬建設との手抜き工事を隠すため、相沢澪を旧体育館東階段から突き落としました。その後、学校だよりと名簿を修正し、転出扱いにしました。

閉校記念サイトは、消された児童の名前をもう一度記録に戻すためのアーカイブでした。
