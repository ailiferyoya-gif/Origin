設備仕様入館管理保全体制

ACCESS CONTROL

入館は、受付・認証・立会の順で管理します。

見学、契約事業者の作業、夜間保守はすべて受付控えを作成します。ラック室への入室には、立会担当者と作業対象の照合が必要です。

VISITOR FLOW

公開エリアと作業エリアを分けています。

受付で本人確認を行い、入館カードを発行します。地下ラック室は、作業票・受付控え・保全台帳の番号が揃う場合のみ入室できます。

  • 夜間入館は事前申請制です。
  • 作業対象ラックは受付で再確認します。
  • 退館時に入館カードと受付控えを照合します。

夜間受付

02:44以降の訂正処理は、受付控えと保全台帳の両方に記録します。
