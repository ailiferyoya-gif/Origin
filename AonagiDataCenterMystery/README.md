# 青凪データセンター DTC-0914

販売想定のモキュメンタリー型Web謎解きです。プレイヤーは、匿名で届いた未整理アーカイブを検証する協力者として調査を開始します。

## 導入
廃止された自治体クラウド移行フォルダから、バックアップ消失案件の監査データが届いた。

依頼内容: 水原梓がなぜ第三サーバ室に入ったのか。消えたバックアップではなく、閉じられた搬入口を追ってください。

## プレイ情報
- 想定時間: 60〜120分
- 推奨人数: 1〜3人
- 入口: `web/index.html`
- 形式: 架空Webサイト横断、資料照合、段階式の調査ノート

## この作品の特徴
IT監査と物理侵入が交差するデータセンター型。ログ名、ラック番号、搬入口が手がかりになる。

## 添付資料
- 公式設備ページ
- 検索キャッシュ
- 保管メール
- 管理画面
- 監査ログ
- 搬入口記録

## 最終的に確定すること
名取啓介が水原梓を第三サーバ室へ誘導し、ラックキーを使って東側搬入口から退路を塞いだ。
