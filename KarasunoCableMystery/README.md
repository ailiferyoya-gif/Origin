# 烏野ケーブル CBL-0209

販売想定のモキュメンタリー型Web謎解きです。プレイヤーは、匿名で届いた未整理アーカイブを検証する協力者として調査を開始します。

## 導入
冬季点検の資料箱から、運休理由と一致しない制動圧ログが見つかった。

依頼内容: 雨宮圭がなぜ山頂折返室に向かったのか。強風ではなく、非常階段扉の扱いを調べてください。

## プレイ情報
- 想定時間: 60〜120分
- 推奨人数: 1〜3人
- 入口: `web/index.html`
- 形式: 架空Webサイト横断、資料照合、段階式の調査ノート

## この作品の特徴
山岳交通の点検記録を読むケーブルカー型。天候情報が見せ札になり、機械ログが本筋になる。

## 添付資料
- 観光案内
- 運行情報
- 点検記録
- 整備会社メモ
- 気象資料
- 制御盤履歴

## 最終的に確定すること
狩谷誠が雨宮圭を山頂折返室へ誘導し、制御盤キーを使って非常階段扉から退路を塞いだ。
