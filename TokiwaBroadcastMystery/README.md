# 常盤放送 BCS-0412

## 作品ジャンル
放送事故にされた告発音声を軸にしたモキュメンタリー型Web謎解き。

## 表向きの依頼
地方局アーカイブの放送事故回を確認する依頼。事故扱いの回だけ素材管理票が欠けている。

## 最初の違和感
素材管理票が欠けており、無音部分の秒数が資料によって微妙に違う。

## 中盤の反転
事故ではなく、数秒だけ告発音声が放送されていた。

## 最終真相
野々村葵は、局上層部とスポンサーの癒着を告発する音声を番組内に紛れ込ませた。局は送信切替ミスとして処理し、素材管理票を抜いた。

## 怖さの出し方
聞こえない音声の怖さ。文字起こし不能、無音、送信切替という事務記録で圧を出す。

## 余韻
復元された告発音声の最後の一語だけが不明瞭。そこに誰の名前が入るのかを想像させる。

## 重要証拠
- 素材管理票
- 無音部分
- 送信切替ログ
- 告発音声
- スポンサー資料
- 野々村葵

## ミスリード
- 単なる放送事故
- 送信切替ミス
- 素材紛失
