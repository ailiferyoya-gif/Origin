# 常盤放送 BCS-0412

販売想定のモキュメンタリー型Web謎解きです。プレイヤーは、匿名で届いた未整理アーカイブを検証する協力者として調査を開始します。

## 導入
地方局アーカイブのデジタル化中、放送事故として処理された回だけ素材管理票が欠けていた。

依頼内容: 野々村葵がなぜ第七中継室に入ったのか。放送事故ではなく、屋上送信機側扉の記録を追ってください。

## プレイ情報
- 想定時間: 60〜120分
- 推奨人数: 1〜3人
- 入口: `web/index.html`
- 形式: 架空Webサイト横断、資料照合、段階式の調査ノート

## この作品の特徴
放送局の素材、送信、屋上導線で読むメディア型。オンエアの表現と裏方ログの差が核。

## 添付資料
- 番組表
- 局内ブログ
- 送信切替ログ
- 素材管理票
- 電波監理資料
- 屋上点検

## 最終的に確定すること
藤堂雅人が野々村葵を第七中継室へ誘導し、副調キーを使って屋上送信機側扉から退路を塞いだ。
