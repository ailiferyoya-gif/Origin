# Nazotoki

Web横断型モキュメンタリー謎解き制作リポジトリです。

GitHub Pagesで公開する場合は、ルートの `index.html` から各作品へ遷移できます。

## 公開入口

- `index.html`: 作品一覧
- `*/web/index.html`: 各作品の入口
- `*_BoothPackage/`: Booth同梱確認用

## 管理方針

- `_backups/`、zip、実行ログ、依存パッケージはGit管理しません。
- `web/logs/` は謎解きコンテンツの一部なのでGit管理対象です。
- 公式サイト修正時は通常版とBooth版を同期します。
