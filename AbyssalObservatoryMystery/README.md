# 黒潮海底観測所 OBS-1118

販売想定のモキュメンタリー型Web謎解きです。プレイヤーは、匿名で届いた未整理アーカイブを検証する協力者として調査を開始します。

## 導入
深海観測ネットワークの旧サーバから、欠測扱いの観測パッケージが匿名で送られてきた。

依頼内容: 三枝透がなぜ第三観測室に残されたのか。欠測ではなく、閉じられた経路を確かめてください。

## プレイ情報
- 想定時間: 60〜120分
- 推奨人数: 1〜3人
- 入口: `web/index.html`
- 形式: 架空Webサイト横断、資料照合、段階式の調査ノート

## この作品の特徴
水圧、音響、欠測ログを積み上げる海底施設型。情報は静かだが、時刻のズレが強く効く。

## 添付資料
- 観測所公式保存データ
- 潮流・音響ログ
- 自治体公開資料
- 保守会社控え
- 研究員メール
- 制限記録

## 最終的に確定すること
榊原怜が三枝透を第三観測室へ誘導し、保守用カードを使って東ケーブルシャフトから退路を塞いだ。
