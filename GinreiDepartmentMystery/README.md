# 銀嶺百貨店 GIN-1224

販売想定のモキュメンタリー型Web謎解きです。プレイヤーは、匿名で届いた未整理アーカイブを検証する協力者として調査を開始します。

## 導入
閉店記念展の準備中、地下フロアの古い防災記録だけが別便で届いた。

依頼内容: 宮永梓がなぜクリスマスイブの地下配送通路にいたのか。閉鎖された売場ではなく、搬入口側を検証してください。

## プレイ情報
- 想定時間: 60〜120分
- 推奨人数: 1〜3人
- 入口: `web/index.html`
- 形式: 架空Webサイト横断、資料照合、段階式の調査ノート

## この作品の特徴
華やかな売場の裏にある物流と防災の謎。表の催事と裏通路の温度差を楽しむ構成。

## 添付資料
- 百貨店公式サイト
- 催事告知
- 防災センター記録
- 配送台帳
- 従業員連絡
- 館内図

## 最終的に確定すること
真壁亮が宮永梓を地下配送通路へ誘導し、搬入口予備鍵を使って旧エレベーター5号機から退路を塞いだ。
