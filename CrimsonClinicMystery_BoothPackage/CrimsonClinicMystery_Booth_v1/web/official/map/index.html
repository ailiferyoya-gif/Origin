院内設備 診療体制 安全管理 院内スタッフ

FACILITY

静けさを保つための院内設計。

診察、処置、記録管理の動線を分け、夜間は一部区画をカード認証で管理しています。

01

第二処置室

予約制処置、点滴、酸素管理に対応します。夜間は通常閉鎖し、入室履歴を保存します。

02

薬品保管庫

処置薬、予備酸素ボンベ、薬剤棚ログを一元管理します。記録の閲覧にはスタッフ認証が必要です。

03

裏口廊下

搬入と夜間対応のための職員導線です。患者さまは通常利用できません。

FLOOR GUIDE

院内の主な区画

受付から待合、診察、処置までの導線を短くし、職員専用区画は患者さまの通路と分けています。

受付・待合
問診票、保険証確認、会計を行います。旧診療所資料の閲覧対応は行っていません。
第一診察室
一般内科、検査説明、処方相談に使用します。
第二処置室
予約処置と夜間対応の区画です。入退室履歴は安全管理記録に保存します。
記録保管棚
診療録と処置記録を分けて保管します。棚卸しは月初に実施します。

RECORD ROOM

受付の奥にある、職員専用の保管室。

紙カルテ、処置記録、薬剤棚ログは同じ部屋で保管しますが、棚番号を分けています。旧診療所資料は入口側の棚ではなく、奥の施錠棚へ移しました。

  • 受付控えは当日分のみカウンター内で保管
  • 処置記録は第二処置室の使用後に回収
  • 旧紙カルテは施錠棚の確認欄へ転記
