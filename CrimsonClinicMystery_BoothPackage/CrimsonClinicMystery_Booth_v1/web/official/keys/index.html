NIGHT PROCEDURE ROUTE

夜間処置の記録手順

処置室の使用後は、薬剤棚の残数、手洗い場の清掃、裏口廊下の施錠を順番に確認します。未処理票が残る場合は、受付ではなく記録保管棚へ戻します。

  1. 01処置記録第二処置室の使用時刻を記入
  2. 02薬剤棚処置薬と予備酸素の残数を確認
  3. 03裏口廊下カード履歴と施錠状態を照合
  4. 04保管棚未処理票と旧紙カルテを戻す
