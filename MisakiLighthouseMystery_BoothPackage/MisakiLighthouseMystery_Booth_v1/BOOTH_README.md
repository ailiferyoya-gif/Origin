# 潮見岬灯台資料館 消灯記録 Booth版

これは架空のWeb横断型モキュメンタリー謎解きです。実在の人物、団体、事件とは関係ありません。

## はじめかた

フォルダ内の `web/index.html` をブラウザで開いてください。

推奨:

- PCブラウザ
- メモを取れる環境
- プレイ時間 3から4時間
- 1から3人プレイ

## 遊び方

灯台資料館、町役場、地方紙、港湾気象台、地域FM、個人ブログ、留言板を巡回し、資料の矛盾を探してください。

答えが分かったら `web/investigate/index.html` の資料照合ノートに入力します。調査ノートは18段階です。

## 内容物

- `web/index.html`: ゲーム入口
- `web/search/`: 検索
- `web/mail/`: メール
- `web/admin/`: 管理画面
- `web/museum/`: 潮見岬灯台資料館
- `web/town/`: 潮見町役場
- `web/news/`: 地方紙風サイト
- `web/weather/`: 港湾気象台
- `web/radio/`: 地域FM資料室
- `web/blog/`: 個人ブログ風サイト
- `web/board/`: 留言板
- `web/investigate/`: 資料照合ノート
- `web/assets/`: ChatGPT生成画像

## 注意

- 画像素材はChatGPT生成画像です。
- スプライトシートは使用していません。
- 被害者は成人です。子供を被害者にする内容は含みません。
- 中級編のため、最初から全サイトへの直リンクはそろっていません。検索語、メール、管理画面を使って調査先を見つけてください。
