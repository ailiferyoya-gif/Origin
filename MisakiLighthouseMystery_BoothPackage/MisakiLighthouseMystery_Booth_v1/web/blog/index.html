2009.10.08 / 展示準備

白灯日誌第4冊を確認。霧笛室の記述は、音の話ばかりで人の名前が少ない。小野寺さんは音だけで扉の向きが分かると言っていた。

2009.10.10 / 港カレー

売店の試食会。関係ないが、閉館式の日はカレーの匂いが展示室まで来そう。第三倉庫の箱は湿気で少し膨らんでいる。

2009.10.12 / 霧

気象台の掲示は南西8m。霧笛室は壁が厚く、外の音が入らない。中の音も、扉が閉じると残るだけになる。

2009.10.12 / 下書き

黒い防水ケースを見つけた。中身は無線記録Cの複写。箱番号はKIRI-1012。第三倉庫へ戻す前に、鍵のことだけ確認する。

この記事は未公開のまま残っていた。本文は短い。「扉は外から閉まる。予備鍵を誰が持っているか。」
