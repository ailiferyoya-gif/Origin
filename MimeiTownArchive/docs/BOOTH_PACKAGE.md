# Booth Package Plan

## Product Shape

Primary sale format: ZIP package.

The buyer downloads the ZIP, extracts it, and opens:

```text
start.html
```

No installation, account, server, or build step should be required.

## Included Files

```text
未明町アーカイブ/
  start.html
  read_me_first.html
  grandmother_letter.html
  notebook/
  sites/
  assets/
  hints/
  endings/
  support/
    how_to_start.html
    troubleshooting.html
    credits.html
    update_history.html
```

## Buyer-Facing Support

### how_to_start.html

Plain instructions:

- unzip the folder
- open `start.html`
- keep files in the same folder
- recommended browser
- how save data works

### troubleshooting.html

Cover common issues:

- page does not open
- browser blocks local storage
- text looks garbled
- progress reset
- audio/image not loading
- smartphone limitations

### hints/

Staged hints for each chapter.
Do not expose all answers on the first click.

## Price Tier Draft

- Standard edition: core ZIP package
- Deluxe edition later: production notes, printable investigation sheet, extra epilogue

Initial suggested price: 800 to 1500 JPY depending on final content volume and art quality.

## Legal And Practical Notes

- All places, people, schools, agencies, and documents must be fictional.
- Avoid names close to real local governments or schools.
- Generated assets should not imitate real news logos, real municipality marks, or real people.
- Include a fiction disclaimer in `credits.html`, but keep it outside the main in-world flow.

## Future Online Version

The same static build can later be hosted under GitHub Pages or another static host.
For paid Booth release, keep the ZIP fully playable offline first.
