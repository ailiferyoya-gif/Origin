# Puzzle Flow

## High-Level Chapters

### Chapter 0: 遺品のUSB

Goal: understand why the investigation starts.
Core action: read the letter, open the notebook, inspect the first three records.
Unlock: first normalized keyword.

### Chapter 1: 日付が合わない

Goal: learn cross-site comparison.
Sites: town, newspaper, school.
Puzzle type: date alignment, event name extraction.
Unlock: library OPAC query.

### Chapter 2: 澪の研究テーマ

Goal: identify what Amamiya Mio was researching.
Sites: library, school, museum.
Puzzle type: call number, club newsletter, exhibition list.
Unlock: `灯台のない灯`.

### Chapter 3: その夜の移動

Goal: reconstruct the disappearance timeline.
Sites: bus, weather, radio, blog.
Puzzle type: timetable logic, tide/weather consistency, transcript keyword.
Unlock: old bus stop and harbor route.

### Chapter 4: 消された町史

Goal: reveal the town had a suppressed local-history incident.
Sites: records, museum, memorial, forum.
Puzzle type: council minutes, family names, forum tripcode, inscription.
Unlock: hidden site path.

### Chapter 5: 灯台のない灯

Goal: solve the final archived truth.
Sites: hidden, notebook, all prior key pages.
Puzzle type: final code assembled from dates, names, coordinates, and phrases.
Unlock: ending pages.

## Answer Input Design

The notebook should normalize:

- full-width / half-width
- hiragana / katakana where appropriate
- common kanji variants
- extra spaces
- simple punctuation

Each answer check should provide:

1. success response
2. near-miss response
3. staged hint link
4. last unlocked location

## Hint Strategy

Hints are sold as part of the product, not an afterthought.
Use three levels:

- Hint 1: where to look
- Hint 2: what to compare
- Hint 3: almost the method, no final answer
- Answer: hidden behind an extra confirm step

## Anti-Dead-End Rules

- Every critical clue must exist in at least two places.
- If a site is needed for progress, the notebook eventually names its category.
- Late-game hidden URLs must be derivable without external search.
- No puzzle should depend on real-world current websites, clocks, social media, or changing search results.

## Ending Structure

### Ending A: Official Record

Player submits a plausible but incomplete answer.
The town's public version is corrected, but the personal truth remains partial.

### Ending B: Archive Restored

Player finds all primary evidence.
Mio's research and Ito's final reason for investigating are preserved.

### Ending C: The Light Remains

Optional true ending.
Player connects the folklore layer, the harbor project, and the preserved names.
This ending should be quiet, not explosive.
