# SOLUTION - 星ヶ浦鉄道 90周年資料 HSR-90

- 時刻表、棚番号、ホーム、スタンプ台紙、掲示板を重ねると「乗れなかった列車」が見える。
- M-20は運休、K-15は回送扱い、月見台の押印時間は空白。つまり、証言時刻に通常の客扱いで月見台へ行く手段がない。
