# Booth販売用 README

## 商品名

星ヶ浦鉄道90周年記念館 - Web横断型モキュメンタリー謎解き

## 内容

架空の鉄道会社、ニュースサイト、自治体サイト、個人ブログ、掲示板を横断して情報を集め、2008年の月見台駅転落事故の真相を解くWeb謎解きです。

## プレイ時間

2から3時間

## 遊び方

1. `web/index.html` をブラウザで開く。
2. ページ内リンクから各サイトへ移動する。
3. 気になる情報をメモする。
4. `資料照合ノート` で15問を順番に解く。
5. 最後の結論まで到達すると真相が表示される。

## 注意

この作品は架空のモキュメンタリー謎解きです。実在の企業、人物、事件とは関係ありません。

## 推奨環境

Chrome、Edgeなどのモダンブラウザ。

ローカルで開く場合は `web/index.html` を直接開けます。リンクや画像がうまく表示されない場合は、同梱の `server.mjs` を使ってローカルサーバーで確認してください。
