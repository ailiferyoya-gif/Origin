# 星ヶ浦鉄道 90周年資料 HSR-90

## 作品ジャンル
記念史から外された駅を軸にしたモキュメンタリー型Web謎解き。

## 表向きの依頼
鉄道90周年記念サイトの年表確認。公式年表と町資料で、ある駅の扱いが違っている。

## 最初の違和感
公式年表と町資料で駅名の扱いが違う。古い時刻表にだけ、現在も使われているような発着時刻が残る。

## 中盤の反転
単なる廃駅ではなく、災害時に切り捨てられた集落の記録だった。

## 最終真相
鉄道会社と町は復旧美談を作るため、救済されなかった駅と住民の記録を記念史から外した。90周年史は地域を救った物語ではなく、切り捨てを隠す編集だった。

## 怖さの出し方
記念サイトの明るさと、古い時刻表の静かな不気味さを対比する。駅が廃止ではなく記載変更されている事務的な怖さを出す。

## 余韻
廃駅の時刻表だけが最新版PDFの隅に残っている。更新ミスとも、誰かが残した痕跡とも取れる。

## 重要証拠
- 公式年表
- 町資料
- 廃駅時刻表
- 復旧美談
- 災害記録
- 記載変更

## ミスリード
- 単なる廃駅
- 広報上の表記ゆれ
- 記念サイトの校正ミス
