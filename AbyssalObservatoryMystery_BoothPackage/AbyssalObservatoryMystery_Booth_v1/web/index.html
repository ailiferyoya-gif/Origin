調査資料 / OBS-1118

調査依頼

黒潮海底観測所 OBS-1118

あらすじ

黒潮海底観測所は、黒潮の縁に沈む海底ケーブルと観測ブイを束ね、沿岸の潮流、圧力、音響を記録していた施設です。閉鎖から数年後、復旧報告書の写しを整理していた資料課の人物から、あなたのもとへ一式の未整理ファイルが届きました。

添えられていた短い依頼文には、同じ夜の出来事が資料ごとに違う名前で処理されている、とだけありました。新聞は研究員死亡を報じ、市の公開文書は休職扱いを残し、保守会社の控えには夜間作業の時刻が残っています。潮流表と音響記録には、閉鎖海域では説明しづらい時刻の記録が混じっていました。

依頼者は、誰かを告発したいのではなく、公式に残された言葉と内部に残った時刻のずれを確かめてほしいと言っています。まずは公開されている公式資料と保存メールを読み、そこから検索画面や管理画面へ進んでください。

調査の仕方

1. 公式資料を読む公開されている案内、管理番号、日付、場所、人物名を確認する。
2. 保存メールや管理画面を確認する略号、時刻、内部処理の控えを見つけ、公開情報との差を控える。
3. 検索画面で外部資料へ進む新聞資料、自治体資料、掲示板保存ログなどは検索語から辿る。
4. 記録を並べる同じ日付、同じ場所、同じ名前が資料ごとにどう扱われているかを見比べる。

確認した内容は調査ノートへ移してください。ひとつの資料だけで判断せず、公開情報と内部記録の差を見ていきます。

調査ノート記録確認と進行管理 公式資料公開されている案内 保存メール内部連絡と控え 検索画面外部資料を探す入口 管理画面制限付き内部記録
