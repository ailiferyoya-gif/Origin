# 潮見岬灯台資料館 消灯記録

Web横断型モキュメンタリー謎解きの中級編です。

## 想定プレイ時間

3から4時間。

## 内容

閉館した潮見岬灯台資料館を中心に、町役場、地方紙、港湾気象台、地域FM、個人ブログ、留言板を横断し、成人の郷土史研究者・綾瀬誠の記録がなぜ「退職扱い」にされたのかを復元します。

被害者は成人です。今後の制作方針として、子供を被害者にしないことを前提にしています。

## サイト構成

- `web/index.html`: 豪華なゲーム説明・導入ページ
- `web/search/`: 検索。検索語で作中サイトを見つけるページ
- `web/mail/`: メール。検索語と管理画面の合言葉の手がかり
- `web/admin/`: 管理画面。合言葉で閲覧制限記録を開く
- `web/museum/`: 潮見岬灯台資料館
- `web/town/`: 潮見町役場 公開資料
- `web/news/`: 岬日日新聞アーカイブ
- `web/weather/`: 岬港湾気象台
- `web/radio/`: しおかぜFM 資料室
- `web/blog/`: 個人ブログ風ページ
- `web/board/`: 港町留言板
- `web/investigate/`: 18問構成の資料照合ノート

入口と照合ノートには全サイトへの直リンクを並べず、検索画面、メール控え、管理画面から段階的にページへ到達する構成です。
