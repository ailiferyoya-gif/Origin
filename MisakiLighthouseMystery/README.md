# 潮見岬灯台資料館 KIRI-1012

販売想定のモキュメンタリー型Web謎解きです。プレイヤーは、匿名で届いた未整理アーカイブを検証する協力者として調査を開始します。

## 導入
灯台資料館の公開写真に、当日の霧笛室だけ説明が欠けている画像が混ざっていた。

依頼内容: 綾瀬誠がなぜ霧笛室へ戻ったのか。濃霧の説明だけで終わらせず、旧灯台側通路を確認してください。

## プレイ情報
- 想定時間: 60〜120分
- 推奨人数: 1〜3人
- 入口: `web/index.html`
- 形式: 架空Webサイト横断、資料照合、段階式の調査ノート

## この作品の特徴
霧、無線、灯台の地形で追う沿岸型。見通しの悪さそのものがミスリードになる。

## 添付資料
- 資料館展示
- 灯台日誌
- 港湾気象台
- 無線記録
- 保守業者控え
- 調査ノート

## 最終的に確定すること
小野寺圭が綾瀬誠を霧笛室へ誘導し、保守鍵を使って旧灯台側通路から退路を塞いだ。
