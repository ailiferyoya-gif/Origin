# 妖札迷宮

縦持ちスマホ向けの和風カード探索RPG MVPです。

## MVP

- 10夜構成の短時間探索
- 召喚でカードを入手
- 20枚ちょうどのデッキを編成してから探索開始
- 高レアカードほど強い性能
- `D:\Codex\WafuNinjaGachaIOSShowcase` を参考にしたスマホ向けガチャUI
- 枠なし/境目なしのカード用スプライトシートを使用
- 3枚の妖札から1枚を選ぶターン制バトル
- 敵の予告行動を見て、攻撃、結界、回復、妨害を選択
- 5夜相当の通常戦闘を経て、10夜でボス戦
- 和風の背景、キャラクター、妖怪、ボス、カード素材はChatGPT生成画像を使用
- キャラクター/妖怪/ボス/カード素材は個別PNG、スプライトシート不使用
