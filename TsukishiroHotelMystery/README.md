# 月白ホテル HTL-1102

## 作品ジャンル
宿泊台帳の身元差し替えを軸にしたモキュメンタリー型Web謎解き。

## 表向きの依頼
閉館整理で出てきた夜間監査室控えの確認。宿泊台帳と監査ログで、ある客の名前が違う。

## 最初の違和感
空室のはずの部屋に清掃記録だけが残っている。夜間監査ログと宿泊台帳の名前が一致しない。

## 中盤の反転
犯人探しではなく、ホテルが客の正体を守るために台帳を差し替えた話だった。

## 最終真相
望月紗季は、追われていた人物を別名で宿泊させた。成瀬亮はホテルを守るふりをして、その人物を追う側に情報を渡していた。

## 怖さの出し方
老舗ホテルの静けさと、空室記録の不自然さで怖くする。ホテル全体が客をいなかったことにできる怖さ。

## 余韻
守られた客の現在は明かさない。匿名依頼人がその客本人なのか、ホテル側の後悔なのかを残す。

## 重要証拠
- 夜間監査室控え
- 宿泊台帳
- 空室清掃記録
- 別名宿泊
- マスターキー
- 望月紗季

- 台帳転記ミス
