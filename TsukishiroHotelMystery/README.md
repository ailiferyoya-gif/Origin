# 月白ホテル HTL-1102

販売想定のモキュメンタリー型Web謎解きです。プレイヤーは、匿名で届いた未整理アーカイブを検証する協力者として調査を開始します。

## 導入
老舗ホテルの閉館整理で、宿泊台帳ではなく夜間監査室の控えだけが匿名で送られてきた。

依頼内容: 望月紗季がなぜ夜間監査室に呼ばれたのか。客室トラブルではなく、従業員階段の移動を調べてください。

## プレイ情報
- 想定時間: 60〜120分
- 推奨人数: 1〜3人
- 入口: `web/index.html`
- 形式: 架空Webサイト横断、資料照合、段階式の調査ノート

## この作品の特徴
ホテルの表向きの接客と夜間監査の裏側を読む宿泊施設型。客室ではなくバックヤードが中心。

## 添付資料
- ホテル公式
- 宿泊台帳
- 金庫開閉ログ
- 従業員メモ
- 館内図
- 監査室控え

## 最終的に確定すること
成瀬亮が望月紗季を夜間監査室へ誘導し、マスターキーを使って従業員階段から退路を塞いだ。
