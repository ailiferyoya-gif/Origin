# 紅坂診療所 CRN-1006

閉鎖された私設診療所に残されたWeb資料を横断し、2010年10月6日の「小さな業務事故」として処理された記録不一致を復元する、上級者向けモキュメンタリー型Web謎解きです。

## プレイ概要

- プレイ時間目安: 3〜5時間
- 難易度: 上級
- 推奨人数: 1〜3人
- 形式: 静的HTMLサイトを調査するブラウザ型謎解き
- 入口: `web/index.html`
- 最終入力: `web/investigate/index.html`

## プレイヤーが調べるもの

- 公式サイト
- 検索アーカイブ
- 保存メール
- 自治体公開文書
- 地方新聞縮刷版
- 業者控え
- 記録保管室
- 古い個人ブログ
- 掲示板保存ログ
- 旧携帯向けページ
- 制限付き管理画面

## 作品の核

この作品の真相は「誰が殺したか」だけではありません。

重要なのは、記録がなぜ別々の場所へ分散され、どの語だけが消され、どの語だけが残ったのかです。プレイヤーは新聞、自治体資料、院内ログ、メール、業者記録を照合し、事故報告書の外側に隠された退路を組み直していきます。

## プレイヤーの立場

プレイヤーは、匿名で提供された紅坂診療所アーカイブの検証協力者です。

依頼人は「事故報告書の外側に残った退路を見つけてほしい」とだけ書き残し、新聞、メール、自治体資料、業者控え、古い掲示板ログを送ってきます。プレイヤーは公開前の資料検証を行う第三者として、記録の矛盾を照合していきます。

## 同梱物

- `HINTS.md`: 段階式ヒント集
- `SOLUTION.md`: 解答・解説・証拠対応表
- `BOOTH_README.md`: 販売ページ用説明文
- `HANDOFF.md`: 制作・運用引継ぎ

## 運営メモ

販売時は `SOLUTION.md` を購入者向けの解答ファイルとして同梱するか、別配布にしてください。公開サイトへ直接リンクしない運用を推奨します。
