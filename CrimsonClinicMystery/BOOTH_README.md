# BOOTH掲載用説明文

## タイトル

紅坂診療所 カルテ改竄事件

## キャッチコピー

閉鎖された診療所の記録は、事件を隠すためではなく、そこへ至る退路を消すために改竄された。

## 商品説明

本作は、架空の診療所「紅坂診療所」に残されたWebサイト、検索アーカイブ、保存メール、自治体資料、新聞縮刷版、業者控え、記録ログを読み解く、モキュメンタリー型Web謎解きです。

プレイヤーは調査用の「照合ノート」に答えを入力しながら、24段階の記録照合を進めていきます。最初は小さな日付や時刻の違和感から始まり、やがてカルテ、鍵台帳、酸素バルブログ、夜間カード、裏口廊下に残された痕跡が一本の線につながります。

派手な演出よりも、作中サイトを実際に調べている感覚、資料の齟齬を見つけた瞬間の納得感、そして最後に「なぜ記録がそう残されたのか」が反転する読後感を重視した作品です。
