# 紅坂診療所 CRN-1006

匿名で届いた未整理アーカイブを読み解き、公式記録と私的記録の食い違いから、隠された真相を復元するWeb謎解きです。

## あらすじ
閉鎖された私設診療所の資料公開前チェックとして、匿名依頼人から事故報告書、保存メール、自治体提出書類、院内ログが届く。

調査を始めると、まず次の違和感が見えてきます。

2010年10月6日の業務事故記録に、カルテ第八冊、酸素バルブログ、薬剤棚ログの欠番がある。死亡記録ではなく「退職扱い」という処理だけが残る。

しかし本当に確認すべきなのは、最初に見える事故や不備ではありません。資料を重ねるほど、医療記録改竄の輪郭が浮かびます。

## プレイ感
- 架空Webサイトを横断して資料を読むモキュメンタリー形式
- 公式資料、ログ、写真、私的メモの差分を照合
- 直接的な恐怖ではなく、記録の食い違いと人間の沈黙でぞくっとする構成

## 推奨環境
スマートフォン、タブレット、PCのブラウザ。インターネット接続は不要です。
