# SOLUTION - 白楼資料館 収蔵箱入替事件 MSM-0817

## 最終解答

MSM-0817 は由来を偽装された持ち出し戻り品。

## 真相

白楼資料館は、観光資源として使える展示品を必要としていた。MSM-0817 は本来、由緒ある祭礼面ではなく、旧館時代に外へ持ち出され、後年戻された由来不明の品だった。だが展示ラベル、台帳、公式サイトの表記が後から整えられ、祭礼資料として扱われるようになった。

奥村志乃は管理画面や校正紙を通じて訂正を進めようとしたが、展示の価値を守りたい側が承認を止めた。プレイヤーは公式情報と裏側の記録の差分を追い、由来偽装の流れを復元する。

## 主要証拠

- 公式収蔵品ページ: MSM-0817 / 由来調査中
- 地方ニュース: 1996/08/19の旧館写真
- 警備ログ: 21:17と21:23の閉館後記録
- メール: 台帳と展示ラベルを先に合わせるという記述
- 台帳: 持ち出し戻り品としての仮登録
- 管理画面: 奥村志乃の更新と空欄の承認欄
- 写真台帳メモ: 借用品、戻すこと
