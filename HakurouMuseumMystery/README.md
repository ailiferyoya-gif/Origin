## プレイヤー導線

匿名の元整理補助員から、展示ラベルと収蔵記録が合わないという依頼が届く。プレイヤーはまず公式サイトで表向きの情報を確認し、検索画面から複数の架空Webページへ移動して、20問の調査ノートを解いていきます。
