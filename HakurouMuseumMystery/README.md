# 白楼資料館 収蔵箱入替事件 MSM-0817

白楼資料館の公式サイト、検索画面、地方ニュース、警備ログ、メール、台帳を横断し、収蔵品 MSM-0817 の由来偽装を追うモキュメンタリー形式のWeb謎解きです。

## プレイヤー導線
匿名の元整理補助員から、展示ラベルと収蔵記録が合わないという依頼が届く。プレイヤーは公式サイトから表向きの情報を確認し、複数の架空Webページを横断して20問の調査ノートを解いていきます。

## 真相
MSM-0817は由緒ある祭礼面として展示されているが、実際には旧館時代に一度外へ持ち出され、後年戻された品だった。資料館側は観光資源化のため由来を整え、台帳と展示ラベルを後から合わせた。奥村志乃はその訂正を進めようとしたが、展示価値を守りたい側に止められた。
