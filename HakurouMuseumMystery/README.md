# 白楼民俗資料館 MSM-0817

## 作品ジャンル
民俗資料の由来偽装を軸にしたモキュメンタリー型Web謎解き。

## 表向きの依頼
収蔵品デジタル化中に見つかった目録番号不一致の調査。目玉展示の説明文だけが複数版存在する。

## 最初の違和感
展示説明文と古い寄贈記録の由来が合わない。展示写真の撮影日と影の向きも微妙にズレる。

## 中盤の反転
展示品の取り違えではなく、村から持ち出された祭具の由来偽装だった。

## 最終真相
資料館設立時、観光資源化のために祭具の出自が改変された。学芸員の奥村志乃はその訂正を進めようとしていたが、展示の価値を守りたい側に止められた。

## 怖さの出し方
展示解説が少しずつ違う怖さ。由来の村名が資料によって変わり、正しい名前を書いた資料だけが隔離されている。

## 余韻
本来の返還先である村は現在の地図に存在しない。行政上消えただけなのか、資料館が消したのかは断定しない。

## 重要証拠
- 目録番号MSM-0817
- 寄贈記録
- 祭具
- 展示説明文
- 奥村志乃の訂正案
- 消えた村名

## ミスリード
- 展示品の取り違え
- 寄贈者名の誤記
- 撮影日の入力ミス
