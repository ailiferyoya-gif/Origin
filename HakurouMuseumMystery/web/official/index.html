01.

Message

白楼町の記憶を、いまに残す。

白楼資料館は、海辺の暮らし、祭礼、古い地名、家々に残された道具を受け継ぐ地域資料館です。小さな館ですが、収蔵品の背景を一つずつ確かめながら、町の記憶を未来へ渡す活動を続けています。

現在、一部資料について受入記録と展示ラベルの照合を進めています。確認中の項目は、確定した表現ではなく暫定情報として公開します。
