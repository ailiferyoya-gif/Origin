# 黒薔薇シネマ CNM-1231

## 作品ジャンル
封印された記録映像を軸にしたモキュメンタリー型Web謎解き。

## 表向きの依頼
閉館上映会パンフレットの上映記録確認。上映リストにない短編フィルムの半券が見つかる。

## 最初の違和感
上映リストにない短編フィルムの半券が残っている。レビューサイトには、その短編を観たという感想が数件だけある。

## 中盤の反転
失われた映画ではなく、劇場内部で撮られた告発映像だった。

## 最終真相
映写技師は、劇場オーナーの不正と関係者の失踪を記録映像に残し、最後の上映で流そうとした。相馬玲奈はそのフィルムを守ろうとしていた。

## 怖さの出し方
存在しない短編のタイトル、半券番号、映写ログを追うほど、観ていないはずの映像が頭に浮かぶ構成にする。

## 余韻
短編フィルムの現物は見つからない。だがレビュー本文だけは妙に具体的で、誰かが本当に観たとしか思えない。

## 重要証拠
- 存在しない短編
- 半券番号
- 映写ログ
- レビュー本文
- 告発映像
- 相馬玲奈

## ミスリード
- 失われた名作映画
- 閉館イベントの隠し上映
- 半券印刷ミス
