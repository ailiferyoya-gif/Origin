# 黒薔薇シネマ CNM-1231

販売想定のモキュメンタリー型Web謎解きです。プレイヤーは、匿名で届いた未整理アーカイブを検証する協力者として調査を開始します。

## 導入
閉館上映会のパンフレット校正中、上映リストと半券束の枚数が合わないことが分かった。

依頼内容: 相馬玲奈がなぜ大晦日の第二映写室にいたのか。上映事故ではなく、非常出口側階段を調べてください。

## プレイ情報
- 想定時間: 60〜120分
- 推奨人数: 1〜3人
- 入口: `web/index.html`
- 形式: 架空Webサイト横断、資料照合、段階式の調査ノート

## この作品の特徴
映画館の閉館と年越し上映を使う劇場型。ロマンチックな表の裏で、半券と映写日誌が刺さる。

## 添付資料
- 劇場公式
- 上映案内
- 映写日誌
- 半券控え
- 閉館ブログ
- 非常口点検

## 最終的に確定すること
久我直人が相馬玲奈を第二映写室へ誘導し、映写室キーを使って非常出口側階段から退路を塞いだ。
