# ナゾトキスキルマップ

謎解き力を鍛えるためのスマホ優先学習アプリです。

## 機能

- 分野別スキルマップ
- 習得度と学習レベル表示
- 暗号、法則発見、言葉遊び、数字推理、観察力の練習問題を各100問、合計500問
- 全スキル0%・Lv.1から開始
- 回答判定、ヒント、解説
- 正答率、連続正解、復習候補、弱点タグの記録
- `localStorage` による学習状況保存

## 起動

`index.html` をブラウザで開くか、任意の静的サーバーで配信します。

```powershell
cd C:\Users\kogit\Documents\Codex\NazotokiTrainer
node server.js
```

起動後は `http://127.0.0.1:4177` を開きます。
