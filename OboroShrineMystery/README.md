# 朧神社 SHR-0107

## 概要
神社の祭礼資料を整理中、奉納帳と祭礼記録で名前の順序が違っている。

## 公開入口で見せる違和感
奉納者名の一人だけ、祭礼当日の記録から消えている。

## 制作メモ
公開ページでは結末を伏せ、調査ノートとSOLUTIONで段階的に明かす。

## 重要証拠
- 奉納帳
- 祭礼記録
- 総代会議メモ
- 土地売却資料
- 古い絵馬
- 消えた奉納者名
