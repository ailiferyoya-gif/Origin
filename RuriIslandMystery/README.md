# 瑠璃島診療船 ISL-0825

## 概要
離島巡回診療船の記念資料から、寄港記録と診療記録が一致しない日が見つかる。

## 公開入口で見せる違和感
寄港していない島の診療記録だけが残っている。

## 制作メモ
公開ページでは結末を伏せ、調査ノートとSOLUTIONで段階的に明かす。

## 重要証拠
- 船舶日誌
- 巡回診療表
- 寄港記録
- 船医控え
- 島名の別表記
- 記念資料
