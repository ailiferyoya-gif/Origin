# 水影ダム DAM-0728

販売想定のモキュメンタリー型Web謎解きです。プレイヤーは、匿名で届いた未整理アーカイブを検証する協力者として調査を開始します。

## 導入
豪雨災害の振り返り資料から、試験扱いの警報ログだけが手書きで訂正されていた。

依頼内容: 伊庭直美がなぜ第二ゲート室に呼ばれたのか。放流判断ではなく、監査廊南扉の記録を見てください。

## プレイ情報
- 想定時間: 60〜120分
- 推奨人数: 1〜3人
- 入口: `web/index.html`
- 形式: 架空Webサイト横断、資料照合、段階式の調査ノート

## この作品の特徴
防災情報と設備運用を読むダム型。自然災害の大きな音の中で、人工的な試験ログが浮く。

## 添付資料
- ダム公式
- 河川情報
- 警報試験ログ
- 巡回記録
- 気象資料
- 管理室メモ

## 最終的に確定すること
葛城学が伊庭直美を第二ゲート室へ誘導し、ゲート鍵を使って監査廊南扉から退路を塞いだ。
