# 水影ダム DAM-0728

## 作品ジャンル
警報時刻の偽装を軸にしたモキュメンタリー型Web謎解き。

## 表向きの依頼
豪雨災害振り返り資料の確認。警報試験ログだけに手書き訂正が残っている。

## 最初の違和感
公式資料、住民メモ、防災無線ログで警報時刻が少しずつ違う。訂正された警報試験ログだけが浮いている。

## 中盤の反転
自然災害そのものではなく、放流判断の遅れを想定外豪雨に見せかけた行政処理が本筋だった。

## 最終真相
管理側は警報発令の遅れを隠すため、試験ログとして時刻をずらした。被害拡大は自然だけのせいではなかった。

## 怖さの出し方
怖いのは水そのものではなく、警報を聞いた人と聞いていないことにされた人の差。社会派の怖さを前面に出す。

## 余韻
住民メモの一つだけ、公式のどの警報時刻とも一致しない。別の警報が鳴っていたのかもしれない。

## 重要証拠
- 警報試験ログ
- 手書き訂正
- 防災無線ログ
- 放流判断
- 住民メモ
- 時刻ずらし

## ミスリード
- 想定外豪雨
- 警報設備の故障
- 住民側の記憶違い
