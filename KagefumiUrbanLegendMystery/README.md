# Kagefumi Urban Legend Mystery

都市伝説「七瀬通りの影踏み電話」を、匿名メッセージ、検索画面、地域ニュース、自治体資料、学校資料、交通会社、店舗、地図、掲示板保存ログを横断して調査する静的Web作品。

## 開始

`web/index.html` から開始。主導線は `web/chat/index.html` と `web/search/index.html`。

## 想定プレイ時間

約3時間。検索語を変えながら複数サイト内の通常ページに紛れた関連資料を見つけ、メッセージに回答して進行する。
