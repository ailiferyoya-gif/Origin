# inheritance-desktop 画像差し替え用プロンプト集

このファイルは、SVGプレースホルダーを後から写真風画像に差し替えるためのプロンプト集です。実在人物、有名人、実在ロゴ、実在企業名は入れないでください。人物の顔は出さず、静かな不穏さとデジタル遺品整理の調査感を優先します。

現在はSVGプレースホルダーを同梱しています。後から写真風画像へ差し替える場合は、同じファイル名で置換してください。Photosのhotspot位置に影響するため、room_0317_0219.svg / room_0317_0219 の木箱位置は「玄関横の傘立ての奥」を維持してください。部屋写真5点は同じワンルーム、同じカメラ位置、同じ家具配置にして、木箱の移動だけが比較で分かるようにします。

SVG版の意図:
- room_0315.svg: 机の上に木箱がある基準写真。
- room_0316.svg: 本棚下段へ木箱が移動した比較写真。
- room_0317_0018.svg: 木箱が見えず、机にスマホと未送信メールらしい紙がある写真。
- room_0317_0219.svg: 傘立て奥に木箱の角が見えるhotspot用写真。
- room_0317_0317.svg: 机の上にスマホだけが残る暗い写真。
- desk_contract_scan.svg: 契約書PDFのスキャン資料。
- mail_header_preview.svg: メールヘッダ矛盾を確認する画面キャプチャ。
- call_waveform_0317.svg: 音声と自動文字起こしの差分を示す復元画面。
- calendar_notification.svg: Calendar通知再生のキャプチャ。
- trash_recovered_files.svg: Trash内の削除済みファイル一覧。
