調査資料 / STG-1120

調査依頼

紅鶴劇場 STG-1120

あらすじ

紅鶴劇場の千秋楽前夜、存在しない演目のチケットが発券された。座席番号は舞台袖を指しており、劇場の公式告知にはその公演だけが掲載されていない。

あなたは興行記録の照合作業を依頼され、座席表、稽古日誌、掲示板の書き込みを調べる。幕が上がる前に、誰が観客として数えられていたのかを見極める。

調査の仕方

1. 公式資料を読む公開されている案内、管理番号、日付、場所、人物名を確認する。
2. 保存メールや管理画面を確認する略号、時刻、内部処理の控えを見つけ、公開情報との差を控える。
3. 検索画面で外部資料へ進む新聞資料、自治体資料、掲示板保存ログなどは検索語から辿る。
4. 記録を並べる同じ日付、同じ場所、同じ名前が資料ごとにどう扱われているかを見比べる。

確認した内容は調査ノートへ移してください。入口にない資料は、検索画面や内部記録の導線から辿ります。
