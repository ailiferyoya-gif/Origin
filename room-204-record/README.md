# 未公開素材 204号室の記録

仮想デスクトップ型のモキュメンタリー謎解きです。

## 起動

`start.html` をブラウザで開くと開始できます。

初期化して開始する場合:

`start.html?reset=1`

## 構成

- `start.html`: 仮想デスクトップ起動ファイル
- `desktop.js`: 進行状態、アプリ、イベント連携
- `desktop.css`: デスクトップUI
- `site/`: KCT番組アーカイブ
- `housing/`: 団地管理サイト
- `archive/`: 素材保管庫
- `social/`: 架空SNS「よりどまり」
- `assets/`: 作品内画像素材
