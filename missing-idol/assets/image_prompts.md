# 写っていない七人目 画像生成プロンプト集

このファイルは、`missing-idol/assets/images/` のSVG/WebP素材を本番用画像へ差し替えるための生成指示です。すべて架空の写真集、架空のアイドルグループ、架空のWeb資料として生成してください。実在アイドル、実在人物、実在グループ、実在雑誌、実在出版社、実在ロゴには似せないでください。

共通設定:
- 架空グループ名: 夜明け前、まだ名前はない。
- 略称: よあなま
- 現行設定: 成人女性6人組
- 消された7人目: 初瀬ミオ。ただし復元後画像以外では本人の姿を明確に出さない
- 現行6人: 全員20歳以上の成人女性。黒髪中心。白ベースの清楚なステージ衣装に、ラベンダー、ペールミント、ペールイエロー、ペールピンク、ホワイトブルー、スカイブルーの差し色
- 7人目の手がかり色: アプリコットベージュ、淡いコーラル
- 露骨な心霊、流血、怪物、ジャンプスケアは禁止
- 日本語文字は崩れやすいため、画像内テキストは少なめにする。重要な謎情報はアプリ側本文でも補完する

## cover_photobook

- ファイル名: `cover_photobook.webp`
- 生成目的: 写真集の表紙。最初は完全に普通のアイドル写真集に見せる
- 構図: 白いスタジオ、自然光、カーテン、花、木の床。成人女性6人がバランスよく並ぶ縦長表紙
- 必須の違和感: なし。怖さを出さない
- 人物・衣装の一貫性: 現行6人の担当カラーを控えめに入れる。全員20歳以上。顔は実在人物に似せない
- 禁止事項: 7人目を写さない。実在雑誌名、実在ロゴ、実在アイドル風の顔を避ける
- ホラー度合い: 0。明るく自然
