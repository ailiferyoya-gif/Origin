# 斑鳩文書館 ARC-0618

## 概要
文書館の寄託資料公開前に、閲覧申請番号だけが欠けた束が見つかる。

## 公開入口で見せる違和感
非公開文書の引用が、公開済み年表に先に現れている。

## 制作メモ
公開ページでは結末を伏せ、調査ノートとSOLUTIONで段階的に明かす。

## 重要証拠
- 寄託文書
- 閲覧申請番号
- 公開年表
- 司書補メモ
- 利用条件書
- 未発行番号
