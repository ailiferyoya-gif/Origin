調査資料 / RST-0629

調査依頼

白浜リゾート RST-0629

あらすじ

白浜リゾートの休館日に、客室清掃済みの通知だけが管理画面へ上がった。予約は入っていないが、ビーチカメラには夜明け前の足跡が映っている。

あなたは宿泊施設の公開資料を確認し、客室案内、清掃ログ、イベント情報を読み比べる。誰も泊まっていない部屋に残った使用痕を追う。

調査の仕方

1. 公式資料を読む公開されている案内、管理番号、日付、場所、人物名を確認する。
2. 保存メールや管理画面を確認する略号、時刻、内部処理の控えを見つけ、公開情報との差を控える。
3. 検索画面で外部資料へ進む新聞資料、自治体資料、掲示板保存ログなどは検索語から辿る。
4. 記録を並べる同じ日付、同じ場所、同じ名前が資料ごとにどう扱われているかを見比べる。

確認した内容は調査ノートへ移してください。入口にない資料は、検索画面や内部記録の導線から辿ります。

調査ノート記録確認と進行管理 公式資料公開されている案内 検索画面外部資料を探す入口 管理画面制限付き内部記録
