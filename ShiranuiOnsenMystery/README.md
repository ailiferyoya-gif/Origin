# 不知火温泉 ONS-0303

販売想定のモキュメンタリー型Web謎解きです。プレイヤーは、匿名で届いた未整理アーカイブを検証する協力者として調査を開始します。

## 導入
温泉地再生プロジェクトの衛生資料に、火災報知とは別系統の燃焼圧ログが挟まっていた。

依頼内容: 三浦千景がなぜ給湯棟裏廊下にいたのか。湯けむりの事故ではなく、露天風呂側通路を見てください。

## プレイ情報
- 想定時間: 60〜120分
- 推奨人数: 1〜3人
- 入口: `web/index.html`
- 形式: 架空Webサイト横断、資料照合、段階式の調査ノート

## この作品の特徴
温泉街の情緒と設備裏を対比する旅館型。湯けむりの雰囲気に隠れた機械ログが要。

## 添付資料
- 旅館案内
- 温泉組合資料
- 燃焼圧ログ
- 衛生点検
- 宿泊者メモ
- 館内図

## 最終的に確定すること
梶原亮が三浦千景を給湯棟裏廊下へ誘導し、機械室札を使って露天風呂側通路から退路を塞いだ。
