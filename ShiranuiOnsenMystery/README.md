# 不知火温泉 ONS-0303

## 作品ジャンル
源泉権と再開発火災を軸にしたモキュメンタリー型Web謎解き。

## 表向きの依頼
温泉地再生プロジェクトの衛生資料確認。火災報知資料に関係ない燃焼圧ログが混ざっている。

## 最初の違和感
旅館案内、再開発資料、源泉権書類で地名が少しずつ違う。旧共同浴場の名前だけが源泉権書類に残る。

## 中盤の反転
旅館火災ではなく、源泉権と土地買収をめぐる隠蔽だった。

## 最終真相
再開発を進めたい側が老舗旅館の設備不良を利用し、火災原因を操作した。三浦千景は源泉権の書類を守ろうとしていた。

## 怖さの出し方
温泉街全体が同じ嘘をついている怖さ。資料ごとに地名が少しずつ変わり、古い名前だけが消えていく。

## 余韻
旧共同浴場の場所は分かるが、現在の地図には載っていない。

## 重要証拠
- 燃焼圧ログ
- 源泉権書類
- 旧共同浴場
- 再開発資料
- 地名の変化
- 三浦千景

## ミスリード
- 旅館火災事故
- 設備点検ミス
- 温泉街の地名整理
