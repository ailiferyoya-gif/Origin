ESCAPE

ホテルから出るために

玄関の自動扉は停電で開かない。非常時案内によれば、発電機室で復旧操作を行い、外扉を手動解錠する必要がある。ただし発電機室へ続く通路は従業員用で、客室側からは二階乾燥室の横を通る。

西階段は、外へ出る道ではない。そこは誰かが「外へ向かった」と見せるために使った場所だ。脱出に必要なのは、西階段の鍵ではなく、乾燥室、従業員通路、発電機室の順序を確かめることになる。

1. 館内図乾燥室と従業員通路の位置を確認する。
2. 客室ログ二十三時四十八分に鍵箱が開いた理由を確認する。
3. 発電機室外扉が本当に開いたかを確認する。
4. 推理外へ出た者がいないなら、失踪は館内で起きたことになる。
館内図乾燥室と通路管理室端末内部記録を見る館内探索拾った物品を確認推理へ進む推理ノート
