吹雪ホテル SNH-1224

あらすじ

奥白嶺の稜線に建つ吹雪ホテルは、十年前の改装を最後に冬季営業を縮小していた。ところが今年の十二月、閉館中のはずの西館から非常灯が点いたという通報が三度続き、役場の保全課へ匿名の封筒が届いた。中には古い客室鍵、焦げたメニューカード、そして「午前零時の客を外へ出さないでください」と書かれた便箋が入っていた。

依頼人は、ホテル創業家の相続整理を担当する司法書士の鳥居真琴。彼女は、失踪した前オーナーの蓮見岳が残した台帳を確認するため、あなたに資料の照合を頼んできた。吹雪で県道は封鎖され、現地に入れるのは保全作業員が開けた短い時間だけ。外へ出る経路を探すことと、蓮見が最後にどこへ向かったのかを確かめることは、同じ作業になる。

ホテルの公開案内は整っている。だが封筒、停電ログ、ルームサービス伝票、宿泊者ノートは、同じ時刻を別々の言い方で記録している。誰かが西階段の時刻を書き換え、誰かが鍵を落とし、誰かが暖炉の灰に紙片を混ぜた。扉を開けるには、鍵の場所だけでなく、消された移動経路を組み直す必要がある。

進め方

1. ストーリーを読むホテルへ入った状況、封筒、停電、午前零時の客を確認する。
2. 脱出手順を組む館内図、封筒、管理室端末から、発電機室へ向かう経路を確かめる。
3. 館内を探索するロビーや西館に残った切り抜き、伝票、宿泊者ノート、静止画を読む。
4. 推理する外へ出る経路と失踪の説明が食い違う場所を、調査ノートで順に整理する。

この作品は吹雪ホテル内で完結します。資料はホテル内の案内、封筒、管理室端末、探索で見つかる物品として扱います。

物語を始めるホテルへ入る 脱出手順発電機室への経路 館内案内地図と非常時案内 封筒依頼文と鍵 推理ノート真相を整理する
