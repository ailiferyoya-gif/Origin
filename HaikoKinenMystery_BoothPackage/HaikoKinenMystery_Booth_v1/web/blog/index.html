六年一組の窓ぎわ
南谷小 平成18年度卒業生の古い日記ログ / since 2006
学校サイト / ニュース / 携帯ページ

2006.07.08 Sat / 晴れ
土曜の校庭

校庭で草取り。軍手に土がいっぱい入った。旧体育館の裏に行くなと先生が言ったけど、理由はよくわからない。
健太くんが、東階段は大人が直すから平気と言っていた。

2006.07.10 Mon / くもり
席替え

窓ぎわになった。山が見える席。給食の時間に牛乳がぬるかった。澪ちゃんは図書室の当番で、雨量の本を探していた。

2006.07.11 Tue / くもり
掲示係

廊下の掲示を貼り替えた。学校だよりの下書きが職員室前に置いてあった。9月の文字が見えたけど、まだ夏休み前なのに早いと思った。
消しゴムをなくした。

2006.07.12 Wed / 蒸し暑い
図書室のカード

図書室の貸出カードに、澪ちゃんの字で「雨量観測ノート」と書いてあった。あの字は丸くてすぐわかる。健太くんは工作クラブの本を借りていた。
どうでもいいけど、この日の冷凍みかんはまだ凍っていた。

2006.07.13 Thu / 雨の前
写真

澪ちゃんが、図書室の箱に旧体育館の写真があると言っていた。階段の板が少し浮いている写真らしい。
「先生に言ったほうがいい」と言ったら、もう言ったけど返事がないと笑っていた。

2006.07.14 Fri / 昼
給食のあと

職員室の前に、帰宅確認の紙がはってあった。番号で並んでいたので、名前はすぐには分からない。
澪ちゃんは昼休みに図書室へ行って、青い表紙のノートを探していた。

2006.07.14 Fri / 雨
黄色い傘

放課後、旧体育館のほうに先生たちが集まっていた。雨が強くて、体育館の裏に水が流れていた。澪ちゃんが図書室で借りたノートを持って、東階段の写真を見せると言っていたのを覚えている。

階段のところに黄色い傘が落ちていた。子ども用の傘だったけど、持っていたのは橘会長だったと思う。健太くんの傘に似ていたから、変に覚えている。

2006.07.14 Fri / 夜
電話

連絡網で「学校の裏には行かないこと」と回ってきた。誰かが帰っていないとか、そういう言い方ではなかった。
でも、母が電話を切ったあと、名前をひとつだけ小さく言った。

2006.07.14 Fri / 23:40
消した下書き

さっき書いたことを一度消した。怖いからじゃなくて、書いたら本当になりそうだったから。
母は「聞いていないことにしなさい」と言った。何を聞いたのか、わたしはまだ聞かれていない。

2006.07.15 Sat / 朝
朝の連絡網

携帯の連絡網に、学校だよりの下書きを差し替えるって出ていた。9月号なのに、日付だけ7月15日で、なんだか変だった。
朝ごはんはトースト。ジャムを落とした。

2006.07.18 Tue / 非公開
パスワード付き記事: 夏休み前のメモ

この記事は当時、クラス内だけに公開されていました。
未入力
澪ちゃんが転校したと先生が言った。誰も挨拶を聞いていない。机は片づけられていたけど、図書室の貸出カードに名前が残っていた。

9月号の学校だよりに転出のことが載るらしい。でも、配られた日付を見たら7月15日になっていた。パスは「展示ケースの箱番号からハイフンを抜いたもの」。

2006.07.16 Sun / 雨あがり
連絡

電話で、学校の裏にはしばらく行かないでと言われた。理由を聞いたら、工事が入るからと言われた。
日記を消そうと思ったけど、忘れるのも変だから残す。

2006.09.01 Fri / 二学期
机

澪ちゃんの机がなくなった。先生は転校と言った。窓ぎわの席だけ、床の色が少し違って見えた。

2006.09.03 Sun / 夜
名前の順番

アルバム委員の紙を見た。名簿は28人、写真は27人。先生は「転出だから」と言ったけど、転出の子の席を片づけた日を誰も覚えていない。
本当は覚えているのかもしれない。覚えていないことにしているだけかもしれない。

2026.04.08 / 閉校後
留言板を見て

閉校サイトのアルバムを見た。やっぱり平成18年度6年1組の写真だけ人数が合わない。あの夏のことは、誰かがちゃんと資料で見つけてくれると思っている。
