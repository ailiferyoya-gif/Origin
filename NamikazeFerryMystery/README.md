# 波風フェリー FRY-0621

販売想定のモキュメンタリー型Web謎解きです。プレイヤーは、匿名で届いた未整理アーカイブを検証する協力者として調査を開始します。

## 導入
港の記念誌に使う写真整理中、旧第二桟橋だけ撮影時刻と入港申告が合わなかった。

依頼内容: 瀬戸真琴がなぜ旧第二桟橋にいたのか。遅延ではなく、機関室側通路の移動を調べてください。

## プレイ情報
- 想定時間: 60〜120分
- 推奨人数: 1〜3人
- 入口: `web/index.html`
- 形式: 架空Webサイト横断、資料照合、段階式の調査ノート

## この作品の特徴
港と船内導線を読むフェリー型。海上の遅延より、桟橋と通路の使われ方に注目する。

## 添付資料
- フェリー公式
- 港湾記録
- 入港申告
- 船内掲示
- 整備メモ
- 乗船案内

## 最終的に確定すること
倉橋徹が瀬戸真琴を旧第二桟橋へ誘導し、保守キーを使って機関室側通路から退路を塞いだ。
