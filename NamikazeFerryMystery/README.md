# 波風フェリー FRY-0621

## 作品ジャンル
非公式寄港と消えた乗客を軸にしたモキュメンタリー型Web謎解き。

## 表向きの依頼
港の記念誌写真と入港申告の照合。旧第二桟橋の写真時刻だけが公式記録と合わない。

## 最初の違和感
旧第二桟橋の写真時刻と入港申告が合わない。公式航路にない寄港の痕跡がある。

## 中盤の反転
単なる遅延ではなく、夜間の非公式寄港が行われていた。

## 最終真相
フェリーは夜間に旧第二桟橋へ非公式寄港し、密輸品と一人の乗客を降ろしていた。瀬戸真琴はその証拠を青い封筒に残した。

## 怖さの出し方
港の夜と、名簿にいない乗客の怖さ。写真の隅に写る記録上いない人でぞくっとさせる。

## 余韻
非公式寄港が一度ではなかったことだけ示す。降ろされた人物のその後は明かしすぎない。

## 重要証拠
- 旧第二桟橋
- 写真時刻
- 入港申告
- 非公式寄港
- 青い封筒
- 乗客名簿

## ミスリード
- フェリー遅延
- 写真の撮影時刻ミス
- 港湾イベントの臨時便
