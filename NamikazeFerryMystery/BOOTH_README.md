# 波風フェリー FRY-0621

匿名で届いた未整理アーカイブを読み解き、公式記録と私的記録の食い違いから、隠された真相を復元するWeb謎解きです。

## あらすじ
港の記念誌写真と入港申告の照合。旧第二桟橋の写真時刻だけが公式記録と合わない。

調査を始めると、まず次の違和感が見えてきます。

旧第二桟橋の写真時刻と入港申告が合わない。公式航路にない寄港の痕跡がある。

しかし本当に確認すべきなのは、最初に見える事故や不備ではありません。資料を重ねるほど、非公式寄港と消えた乗客の輪郭が浮かびます。

## プレイ感
- 架空Webサイトを横断して資料を読むモキュメンタリー形式
- 公式資料、ログ、写真、私的メモの差分を照合
- 直接的な恐怖ではなく、記録の食い違いと人間の沈黙でぞくっとする構成

## 推奨環境
スマートフォン、タブレット、PCのブラウザ。インターネット接続は不要です。
