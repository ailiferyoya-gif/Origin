# 波風フェリー FRY-0621

匿名で届いた未整理アーカイブを開き、公開資料・ログ・個人メモを照合して、記録の奥に隠された経路を突き止めるWeb謎解きです。

## あらすじ
港の記念誌に使う写真整理中、旧第二桟橋だけ撮影時刻と入港申告が合わなかった。

「瀬戸真琴がなぜ旧第二桟橋にいたのか。遅延ではなく、機関室側通路の移動を調べてください。」

## 遊び方
- `web/index.html` から開始します。
- 各資料ページを読み、気になる日付・場所・人名・ログ名をメモします。
- 調査ノートで回答を照合すると次の段階が開きます。

## 推奨環境
スマートフォン、タブレット、PCのブラウザ。インターネット接続は不要です。
