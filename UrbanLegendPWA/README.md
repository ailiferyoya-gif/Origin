# 赤いワンピースの少女

iPhone Safariで遊べる都市伝説ミステリーPWAです。

## 現在の版

中身確認用の長編調査版です。BOOTH販売準備ではなく、約3時間プレイを目指すための本編構造確認を優先しています。

## 追加された遊び

- 6章構成
- 12個の小謎
- 章ロックと順次解放
- 3段階ヒント
- 証拠9点
- 検索結果8件
- 現在目標の常時表示

## 遊び方

- `index.html` をブラウザで開くとプレイできます。
- ローカルサーバーで開く場合: `http://127.0.0.1:4177/index.html`
- チャット、検索、証拠、推理の4タブを行き来して進めます。

## 素材方針

画像素材はChatGPT生成画像のみを使用しています。キャラクター画像とスプライトシートは使用していません。
